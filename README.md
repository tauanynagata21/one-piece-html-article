# One Piece Article - HTML Practice

Projeto desenvolvido para praticar estrutura de texto em HTML utilizando div, span, listas e títulos.

## Objetivo
Aprender a organizar textos longos em HTML de forma estruturada.

## Elementos utilizados
- div
- span
- strong
- p
- ul e li
- headings (h1, h2)

## Tecnologias
- HTML5

## Observação
Projeto de estudo inicial em estruturação de conteúdo web.
